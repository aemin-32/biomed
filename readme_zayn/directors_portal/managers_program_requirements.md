# متطلبات ومميزات برنامج الإدارة العليا (Hospital Directors Portal)

هذا الملف يوضح الرؤية الشاملة للبرنامج المنفصل الخاص بمدراء المستشفى أو صناع القرار. 
الهدف من هذا البرنامج ليس عرض التفاصيل الفنية المعقدة للصيانة، بل عرض **مؤشرات الأداء (KPIs)**، **التكاليف المالية**، و**جاهزية المستشفى**.

## 1. لوحة القيادة الرئيسية (Executive Dashboard)
- **مؤشر الجاهزية (Uptime/Downtime):** نسبة الأجهزة التي تعمل بكفاءة مقابل الأجهزة المعطلة.
- **التنبيهات الحرجة:** الأجهزة الاستراتيجية المعطلة (مثل الرنين المغناطيسي، أجهزة التنفس) وتأثيرها على المرضى.
- **نظرة مالية سريعة:** إجمالي تكاليف الصيانة لهذا الشهر مقارنة بالميزانية المحددة.

## 2. إدارة الأصول والممتلكات (Asset Management)
- **قيمة الأصول:** إجمالي القيمة المالية للأجهزة الطبية الموجودة في المستشفى.
- **العمر الافتراضي والإهلاك (Depreciation):** تنبيهات للأجهزة التي تجاوزت عمرها الافتراضي وتحتاج إلى استبدال (بناءً على تاريخ التركيب وتكلفة الشراء).
- **توزيع الأجهزة:** خريطة حرارية أو رسوم بيانية توضح توزيع الأجهزة على الأقسام المختلفة.

## 3. التقارير المالية (Financial Reports)
- **تكاليف الصيانة:** مقارنة تكلفة إصلاح الجهاز مقابل تكلفة شراء جهاز جديد.
- **استهلاك المخزون:** تقارير عن أكثر الأقسام استهلاكاً لقطع الغيار والمواد الطبية.
- **عقود الصيانة (SLAs):** متابعة الشركات الموردة ومدى التزامها بعقود الصيانة الدورية (PM).

## 4. مؤشرات أداء الكادر (Staff Performance KPIs)
- **سرعة الاستجابة (Response Time):** متوسط الوقت الذي يستغرقه المهندسون للاستجابة لبلاغات الأعطال.
- **وقت الإصلاح (Resolution Time):** متوسط الوقت لإصلاح الأعطال وإعادة الجهاز للخدمة.
- **إنتاجية المهندسين:** عدد المهام المنجزة لكل مهندس.

## 5. الصلاحيات (Permissions)
- **قراءة فقط (Read-Only):** في الغالب، يحتاج المدراء إلى صلاحيات "قراءة فقط" للبيانات، مع إمكانية تصدير التقارير (PDF/Excel).
- **الموافقات (Approvals):** إمكانية الموافقة على طلبات الشراء باهظة الثمن التي يرفعها المهندسون أو أمناء المخازن.
